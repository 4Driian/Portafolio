Portfolio

Dalio Project

Dalio was an exciting project developed during the first semester of Web Design. Its main focus was the creation of a unique and appealing website template. The main objective of this project was to allow us to acquire and put into practice valuable skills in the application of styles and web design.

Biccas Project

Biccas was an engaging project undertaken during the second semester of Web Design. It revolved around the development of a captivating and visually appealing website template. The primary goal of this project was to provide us with the opportunity to learn and apply valuable skills in the realm of styling and web design. Through the Biccas project, we aimed to enhance our proficiency in creating dynamic and user-friendly web interfaces, as well as effectively implementing responsive design principles.

BenitoShop Project

Benito Shop is the result of a real entrepreneurial venture that emerged in 2020 with the genuine intention of establishing an authentic online commerce. Initially created for illustrative purposes as a final project, as it progressed, the viability and potential of the business became evident. The idea of turning it into a real e-commerce has become an exciting and concrete possibility.

Dad Jokes Project

The objective of this project was to create a dad joke generator that allows users to search for jokes using keywords. Additionally, we implemented a shopping cart feature for custom objects with dad jokes. It was a real challenge to start with because we began incorporating APIs.

CETAV Project

I've been part of the migration of the CETAV website, which was filled with challenges, working with new frameworks for me. It didn't pose an extreme difficulty, and working closely with my team, we managed to achieve a very good result.
